# ♟️ Simulador de Xadrez em C

## 📌 Descrição
Este projeto consiste na simulação de movimentos de peças de xadrez utilizando a linguagem C. O objetivo é aplicar conceitos fundamentais de lógica de programação, como estruturas de repetição, condicionais e organização de código.

## 🚀 Tecnologias utilizadas
- Linguagem C

## 💡 Funcionalidades
- Simulação do movimento das peças:
  - Torre (movimento linear)
  - Bispo (movimento diagonal)
  - Rainha (movimento livre)
  - Cavalo (movimento em L)
- Diferentes níveis de complexidade:
  - Novato: uso básico de loops
  - Aventureiro: loops aninhados
  - Mestre: uso de recursividade e lógica avançada

## 📚 Conceitos aplicados
- Estruturas de repetição (`for`, `while`, `do-while`)
- Condicionais
- Loops aninhados
- Recursividade
- Organização e legibilidade de código
